# Evaluation Framework Documentation

Central index for the 24-task Eval AI Agent portfolio. Task **deliverables** (code, reports, configs) live in each task folder under `beginner/`, `intermediate/`, `advanced/`, and `devops/`.

| File | Description |
| ---- | ----------- |

| [ORCHESTRATOR_DESIGN.md](ORCHESTRATOR_DESIGN.md) | Task router/executor architecture |

## Eval tooling (compare + dashboard — any clone)

```bash
make eval-api          # START THIS — http://127.0.0.1:8788 live dashboard
make eval-bot TASK=B3  # single bot
make eval-bots-all     # all 24 bots
```

| For | Use |
| --- | --- |
| Human docs | [`AGENT_API.md`](AGENT_API.md) |
| Agent: task instructions | `GET /api/agent/guide/{TASK}` |
| Agent: submit work | `POST /api/agent/submit` |
| Portfolio JSON | `GET /api/portfolio` |

```bash
make eval              # verify deliverables only
make eval-compare TASK=B2 AGENT_OUTPUT=./out.csv
make eval-metrics      # Prometheus text
```

Local agent runs stored under `.eval/` (gitignored). `docs/eval-status.json` is generated by `make eval`.
